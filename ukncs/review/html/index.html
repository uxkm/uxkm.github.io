선언방식?

  • HTML 4.01/XHTML 1.0 Transitional DTD

Doctype

  • 웹문서 종류를 알려주는 선언문.
  • DTD로 정의.
  • 웹문서 처음에 선언.
  • 선언X 호환모드(Quirks Mode)로 화면표시.

학습 내용

  • HTML이 무엇인지 알 수 있음.
  • HTML의 기본구조를 배움으로 기초를 시작.
  • HTML에서 사용하는 요소에 대해 알아가기.
  • HTML 기본 레이아웃(문서구조)를 배움.

HTML이란?

  • HyperText Markup Language의 약자
  • 월드와이드웹(WWW)의 기초적인 구성요소
  • 웹페이지 표준 마크업 언어
  • 문서라고도 불리며, HTML 요소들로 구성
  • 웹 페이지의 디자인이나 기능을 결정하는데 사용
  • 웹콘텐츠의 의미와 구조를 정의

HTML 기본 구조

  • <!DOCTYPE html> : 문서가 HTML5 임을 명시
  • <html> : 루트(root) 요소를 정의
  • <title> : 제목(title)을 정의
  • <body> : 문서의 내용(Content)

HTML 요소

블록 요소

  • 새로운 라인(line)에서 시작
  • 해당 라인의 모든 너비를 차지
  • div, h1~h6, header, section, article, footer 등

인라인 요소

  • 내용(content)만큼만 차지
  • span, a, strong, b, em, i 등

HTML5 구분

  • 콘텐츠모델별로 분류
  • 각 요소들은 특성에 따라 분류.

콘텐츠 모델

메타데이터 콘텐츠

  • 웹 문서와 관련된 정보를 표현하는
  • 콘텐츠와 다른 문서와의 관계를 유지
  • 콘텐츠의 프리젠 테이션 또는 동작을 설정
  • 정보를 전달하는 콘텐츠
  • head안에 포함되는 요소

base, link, meta, noscript, script, style, template, title

  • 대분분 body요소 안에서 사용하는 요소들

섹션 콘텐츠

  • 웹 문서의 섹션 영역을 정의
  • 헤딩과 푸터의 유효범위를 지정
  • 제목과 그 내용을 포함하는 범위를 지정
  • Block Elements

article, aside, nav, section

헤딩 콘텐츠

  • 섹션 콘텐츠 or 암시적으로 마크업 된 영역을 정의
  • Block Elements

h1, h2, h3, h4, h5, h6

프레이징 콘텐츠

  • 문장과 텍스트가 관련된 요소
  • 단락 내부레벨에서 마크업을 하는 요소
  • 프레이징 콘텐츠가 모여 문단을 구성
  • Inline Elements

임베디드 콘텐츠

  • 다른 소스를 가져오거나 삽입시키는 컨텐츠와 관련

  • 콘텐츠를 포함하고 있지만 메타데이터가 아닌 것(SVG등)
  • 외부 리소스가 사용이 불가능 할때 사용할 대체 콘텐츠
  • Inline Elements
