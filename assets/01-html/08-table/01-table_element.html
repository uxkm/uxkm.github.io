요소 설명

HTML Table 요소는 (<table>) 2차원 이상의 데이터를 나타냅니다.
테이블에는 자손에는 행, 열 및 셀이 있습니다. 행과 열은 그리드를 형성합니다.

  1. 표는 <table> 태그로 만듭니다.
  2. 행(가로 한 줄)은 <tr> 태그로 만듭니다.
  3. 열의 제목이 들어가는 셀은 <th> 태그로 만듭니다.
  4. 내용이 들어가는 셀은 <td> 태그로 만듭니다.
  5. 각 열의 의미에 따라 <thead>, <tbody>, <tfoot> 태그로 구분지을 수도 있습니다.
  6. 가로로 이웃한 셀을 합칠 때에는 colspan 속성을 사용합니다.
  7. 세로로 이웃한 셀을 합칠 때에는 rowspan 속성을 사용합니다.
  8. 표 제목은 <caption> 태그로 만듭니다.
  9. deprecated(더 이상 사용되지 않음)속성은 CSS 속성을 사용

    align - CSS 속성값으로 사용

    bgcolor - CSS 속성값으로 사용

    border - CSS 속성값으로 사용

    cellpadding - CSS 속성값으로 사용

    cellspacing - CSS 속성값으로 사용

    frame - CSS 속성값으로 사용

    rules - CSS 속성값으로 사용

    width - CSS 속성값으로 사용

  10. summary - 테이블을 직접 확인할 수 없는 경우에 대비하여
    해당 테이블을 설명하기 위한 대체 텍스트 사용을 정의합니다.
    일반적으로 점자 화면을 사용하여 웹을 탐색하는 사람들이 정보를 얻을 수 있도록 설명합니다.
    더 이상 사용되지 않는 속성으로, 다음과 같이 대체할 수 있습니다.

    테이블의 <caption>요소를 사용.

    <details>내부 요소 <caption>요소를 사용.

    <figure>요소에 <table>요소를 포함시키고 설명을 추가.

    <figure>안에 <figcaption>요소를 사용하여 설명을 추가.
