요소 설명

HTML <caption> 요소 (or HTML Table Caption Element)는 표의 제목을 나타냅니다.
이 요소는 반드시 <table>의 첫 번쨰 자식 요소로 배치되어야 하지만, CSS를 사용한 styling은 표와 관련된 다른 곳에 배치할 수 있습니다.
summary 속성 대신 대체 텍스트 사용을 정의하며, 일반적으로 점자 화면을 사용하여 웹을 탐색하는 사람들이 정보를 얻을 수 있도록 설명합니다.
