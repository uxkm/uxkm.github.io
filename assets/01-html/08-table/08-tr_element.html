요소 설명

<tr> 요소는 테이블에서 셀들로 이루어진 하나의 행(row)을 정의할 때 사용합니다. <tr> 요소는 하나 이상의 <th> 요소(header cell)나 <td> 요소(data cell)를 포함할 수 있습니다.

HTML5에서는 <tr> 요소의 align, bgcolor, char, charoff, valign 속성을 더 이상 지원하지 않습니다.

CSS 기본값 : display:table-row; border-color:inherit; vertical-align:middle;
