요소 설명

<object> 요소에 의해 호출되는 플러그인의 매개 변수(parameter)를 정의할 때 사용합니다.
단일 요소이며, HTML에서는 <param> 요소를 닫지 않지만, XHTML에서는 반드시 태그를 닫아야만 합니다.

HTML5에서는 <param>요소에서 type, valuetype 속성을 더 이상 지원하지 않습니다.

CSS 기본값 :

6.13.1. param 속성

[name] <param name="이름">

매개변수의 이름을 명시합니다. 이 속성은 value 속성과 함께 사용하여 <object> 요소에 의해 호출되는 플러그인의 매개변수(parameter)를 명시합니다. name 속성값에는 명시된 객체가 지원하는 이름이라면 어떠한 이름이라도 사용할 수 있습니다.

[value] <param value="값">

매개변수의 값을 명시합니다. 이 속성은 name 속성과 함께 사용하여 <object> 요소에 의해 호출되는 플러그인의 매개변수(parameter)를 명시합니다. value 속성값에는 명시된 객체가 지원하는 값이라면 어떠한 값이라도 사용할 수 있습니다.
