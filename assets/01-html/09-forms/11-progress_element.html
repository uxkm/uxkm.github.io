요소 설명

<progress> 요소는 작업의 진행률을 나타낼 때 사용합니다.
이러한 <progress> 요소는 자바스크립트와 함께 작업의 진행 정도를 표현할 때 사용되며, 범위를 나타내는 데 사용하기에는 적합하지 않습니다.
범위를 나타낼 때는 <progress> 요소 대신 <meter> 요소를 사용해야 합니다.

<progress> 요소는 HTML5에서 새롭게 추가된 요소입니다.
Internet Explorer 10+ 부터 지원합니다.

9.11.1. progress 속성

[max] <progress max="숫자">

<progress> 요소의 max 속성은 작업이 모두 완료되기까지 필요한 총 작업량을 명시합니다.
작업이 모두 완료되기까지 필요한 총 작업량을 부동 소수점 수(floating point number)로 명시합니다.
기본값은 '1.0'입니다.

[value] <progress value="숫자">

<progress> 요소의 현재까지 완료된 작업량을 부동 소수점 수(floating point number)로 명시합니다.
