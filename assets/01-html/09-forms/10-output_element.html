요소 설명

<output> 요소는 스크립트 등에 의해 수행된 계산의 결과나 사용자의 액션에 의한 결과를 나타낼 때 사용합니다.

<output> 요소는 HTML5에서 새롭게 추가된 요소입니다.
Internet Explorer는 지원하지 않습니다.

CSS 기본값 :

9.10.1. output 속성

[name] <output name="이름">

name 속성은 <output> 요소의 이름을 명시합니다.
name 속성은 폼(form)이 제출된 후 서버에서 폼 데이터(form data)를 참조하기 위해 사용되거나 자바스크립트에서 요소를 참조하기 위해 사용됩니다.

[for] <output for="요소 id">

for 속성은 계산 결과와 계산에 사용되는 요소 사이의 관계를 명시합니다.

[form] <output form="form id">

<output> 요소가 포함될 하나 이상의 <form> 요소를 명시합니다.
이 속성값은 반드시 같은 문서 내에 존재하는 <form> 요소의 id 속성값이어야 합니다.
