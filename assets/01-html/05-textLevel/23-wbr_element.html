요소 설명

<wbr> 요소(Word Break Opportunity)는 단어 중간에서 행바꿈 될 수 있는(line-break) 위치를 정의할 때 사용합니다.
길이가 굉장히 긴 단어의 경우 브라우저가 해당 단어를 잘못된 위치에서 행바꿈 하지 않도록 <wbr> 요소를 사용하여 행바꿈 될 수 있는 위치를 직접 명시할 수 있습니다.

<wbr> 요소는 HTML5에서 새롭게 추가된 요소입니다.
Internet Explorer : No support(지원하지 않음)
